Trailers4Jellyfin

Help

Downloads trailers for upcoming and recently released movies not in your library from TMDB/YouTube, then plays them before your movies — no Cinema Mode plugin needed.

A free TMDB API key is required. Get one here →

TMDB
Your TMDB Read Access Token (JWT) or v3 API key from themoviedb.org/settings/api.

Cinema Mode

When enabled, this plugin plays trailers from your download folder before each movie.
Two steps required:
1. Add the Download Folder below as a Jellyfin Movies library (e.g. named "Trailers") and run a library scan.
2. Enable Cinema Mode in each user's Jellyfin playback settings (User settings → Playback → Cinema mode → On).

Registers this plugin as a trailer intro provider with Jellyfin.
How many trailers to play before each movie starts. Default: 2.
Prefer trailers whose genre overlaps with the movie you're about to watch (e.g. horror trailers before horror films, family trailers before animated movies). Falls back to random trailers if not enough genre matches are found.
Uses each Jellyfin user's watched status so users don't have to rewatch the same intro trailers.
Movie starts per user. 1 = every movie. 3 = show on the first movie, skip the next 2, then show again on the 4th movie.
Avoids showing trailers when a user resumes a partially watched movie.

Trailer Languages

Only download trailers in the selected languages. Leave everything unchecked to allow all languages.

Trailer Rotation

Keep your trailer library fresh by automatically removing old or watched trailers each time the download task runs.

Maximum number of trailers to keep on disk at once. When this limit is exceeded, the oldest trailers are deleted first to make room for new downloads. Set to 0 for unlimited. Default: 50.
Trailers that any user has already watched will be deleted at the start of each download run, keeping the pool fresh.

Advanced
Full path to yt-dlp. Recommended — yt-dlp is far more reliable than the built-in downloader and handles YouTube bot protection automatically. Leave blank to auto-detect from common locations and PATH, or fall back to the built-in downloader (720p max) if yt-dlp is not found.
Full path to ffmpeg. Required by yt-dlp to merge separate audio and video streams into a single .mp4 (needed for 1080p and when yt-dlp cannot find ffmpeg automatically). Leave blank if ffmpeg is on the system PATH.
Path to a cookies.txt file (Netscape format) exported from a browser while logged into YouTube. Fixes "VideoUnavailableException" errors that occur when YouTube blocks automated requests from server IPs. Use a browser extension such as "Get cookies.txt LOCALLY" (Chrome/Firefox) to export the file, then copy it to your Jellyfin server and enter the path here. Works with both the built-in downloader and yt-dlp.
