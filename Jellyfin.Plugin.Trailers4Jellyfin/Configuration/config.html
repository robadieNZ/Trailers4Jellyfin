Trailers4Jellyfin

Help

Automatically downloads movie trailers from TMDB/YouTube and saves them locally so Jellyfin Cinema Mode can play them before your movies. Trailers are downloaded by a scheduled task that runs daily (configurable via Scheduled Tasks in the dashboard).

A free TMDB API key is required. Get one here →

TMDB Settings
Your TMDB v3 API key. Found at themoviedb.org/settings/api.

Download Location
When enabled, each trailer is saved into a trailers/ subfolder next to the movie file (e.g. The Matrix (1999)/trailers/The Matrix-trailer.mp4). Jellyfin automatically picks these up as Local Trailers, which works seamlessly with Cinema Mode. Recommended.
Folder where trailers will be saved when not placing them alongside movies. You can point a Jellyfin Movies library at this folder and use it as a Pre-Roll library in Cinema Mode.

Download Options
How many trailers to download per movie. Default: 1.
The built-in downloader (YoutubeExplode) supports up to 720p without any extra tools. For 1080p, configure yt-dlp below.
Skips any movie that already has a local trailer in Jellyfin or in its trailers/ folder. Enabled by default.

Advanced
Full path to the yt-dlp executable. When set, yt-dlp is used instead of the built-in downloader, enabling 1080p quality. Requires ffmpeg on your system PATH for audio/video merging. Leave blank to use the built-in downloader (720p max, no extra tools).
